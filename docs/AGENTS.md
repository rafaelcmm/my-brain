# AGENTS

## Scope

Applies to docs/ user and technical documentation.

## Responsibilities

1. Keep docs split by audience: user in root README, technical in docs/.
2. Keep examples copy-paste ready.
3. Keep troubleshooting tied to observable symptoms.

## Change Constraints

1. Any changed command path must be reflected in quickstart and runbooks.
2. Keep versioning and security sections current with release workflow.
